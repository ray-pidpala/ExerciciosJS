JavaScript

JavaScript é uma linguagem criada na década de 1990 voltada para a parte de desenvolvimento web. É uma linguagem do lado do cliente (cliente-side) e possui tipagem dinâmica, ou seja, o tipo da variável é atribuido automaticamente
